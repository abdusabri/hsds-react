# AvatarSelector

An AvatarSelector can be used as a custom renderTrigger for a DropdownV2.

## Example

```jsx
<AvatarSelector />
```

## Props

| Prop   | Type      | Description                                 |
| ------ | --------- | ------------------------------------------- |
| image  | `string`  | URL of the image.                           |
| isOpen | `boolean` | Used to control the direction of the caret. |

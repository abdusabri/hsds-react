# EditableTextarea

A EditableTextarea does something special.

## Example

```jsx
<EditableTextarea />
```

## Props

| Prop      | Type       | Default | Description                     |
| --------- | ---------- | ------- | ------------------------------- |
| className | `string`   |         | The className of the component. |
| innerRef  | `Function` |         | Retrieve the inner DOM node.    |
